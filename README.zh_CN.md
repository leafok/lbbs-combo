# LBBS-combo - LeafOK BBS 所有组件的组合包

简介
=================
本软件包提供了一个预配置的基于 Docker 的 LeafOK BBS 运行环境，包含 Web 版本和 Telnet 版本，适用于测试或演示目的。

安装
==================
您可以通过以下两种方式之一进行安装：

1. 从源代码构建 Docker 镜像：  
   sh -x build.sh  

2. 从 Docker Hub 拉取预构建的镜像：  
   docker compose pull  

启动命令：  
   docker compose up -d
